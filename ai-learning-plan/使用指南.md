### 2. 信息采集流程
点击"开始定制学习方案"进入6步骤信息采集：

#### 步骤1：性别选择
- 男性 / 女性 / 不愿透露
- 单选卡片式设计

#### 步骤2：身份选择
- 职场人士 / 大学生 / 高中生
- 根据身份调整学习内容

#### 步骤3：母语选择
- 中文 / 英语 / 阿拉伯语 / 其他
- 影响学习难度曲线

#### 步骤4：学习目标
- 文本输入，10-500字符
- 描述具体学习需求

#### 步骤5：时间安排
- 每日学习时间：30-120分钟
- 每周学习天数：3-6天
- 目标完成时间（可选）

#### 步骤6：基础评估
- 零基础/有基础判断
- 自评英语水平（A1-B2）
- 影响起点难度设置

## 🔧 技术架构

### 核心技术
- **Next.js 14**：React全栈框架
- **TypeScript**：类型安全保障
- **TailwindCSS**：现代化样式框架
- **Lucide React**：优雅的图标库

### 组件架构
```
src/
├── app/                    # 页面路由
│   ├── page.tsx           # 首页
│   └── wizard/page.tsx    # 信息采集
├── components/
│   ├── ui/                # 基础组件
│   │   ├── button.tsx
│   │   ├── card.tsx
│   │   └── input.tsx
│   └── wizard/            # 向导组件（简化版）
└── lib/                   # 工具库
```
